## Projects

1. **ToDo List App**  
   - Add and delete tasks  
   - Clean layout with styled containers  

2. **Guess The Number Game**  
   - User enters a number guess  
   - Shows feedback (Right/Wrong)

3. **Booksy (Book Management App)**

   - Delete books from the list  
   - Simple and clean UI

- JavaScript
